About the Project

This is a simple and effective website designed to assist university professors and instructional designers. The tool generates structured and optimized Artificial Intelligence prompts focused on creating teaching materials, lesson plans, and multimedia resources.

Features

Interactive Prompt Generator: Fill-in-the-blanks inputs to customize student context (course, profile, class, language).

Copy & Paste: Integrated button to copy the formatted prompt directly to the clipboard.

AI Tools Menu: A dedicated page listing recommended tools (Invideo, Prezi, Murf, etc.) with direct links.

Clean Interface: Responsive design using Bootstrap 5 with a professional blue visual identity.

How to Use

Open the index.html file (or the configured homepage).

Fill in the blank fields with your course and student information.

Click "Copy Prompt" (Copiar Prompt).

Paste the text into an AI (like Gemini or ChatGPT) to start planning your lesson.

Use the "Continue" button to view the list of suggested tools.
